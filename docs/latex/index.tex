\hypertarget{index_intro}{}\doxysection{\texorpdfstring{Introdução}{Introdução}}\label{index_intro}
O presente trabalho tem como objetivo desenvolver um programa em linguagem C capaz de criar e manipular vetores de números inteiros. O sistema permite ao utilizador realizar diversas operações, como ordenação, cálculo da soma e média dos elementos, bem como operações entre dois vetores distintos. Para facilitar a interação, o programa disponibiliza um menu intuitivo que orienta o utilizador na execução das funcionalidades. Além das operações básicas, o projeto foi concebido com uma estrutura modular, garantindo organização e facilidade de manutenção do código. Esta abordagem contribui para a clareza e eficiência do programa, permitindo futuras expansões e melhorias. O trabalho visa consolidar os conhecimentos adquiridos na disciplina, aplicando conceitos fundamentais de programação estruturada e boas práticas de desenvolvimento.\hypertarget{index_features}{}\doxysection{\texorpdfstring{Funcionalidades}{Funcionalidades}}\label{index_features}

\begin{DoxyItemize}
\item Leitura e escrita de vetores
\item Ordenação e cálculo de mediana
\item Identificação de números primos
\item Construção e transposição de matrizes 18x18
\item Calculo da raíz quadrada
\item Devolução do vetor, com a primeira metade dos valores ordenada por ordem crescente
\item Cálculo da soma da primeira metade dos elementos no vetor com os da segunda metade (dá um vetor com metado do tamanho)
\item Devolução dos valores do vetor que não são divisíveis por três
\item Devolução de um vetor que mistura metade do primeiro vetor e metade do segundo
\item Cálculo e apresentação da matriz transposta
\item Apresentação de uma página de ajuda
