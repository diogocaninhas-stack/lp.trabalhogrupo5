Trabalho de Laboratorio de Programacao - Grupo 5(PL4)

Introdução

O presente trabalho tem como objetivo desenvolver um programa em linguagem C capaz de criar e manipular vetores de números inteiros. O sistema permite ao utilizador realizar diversas operações, como ordenação, cálculo da soma e média dos elementos, bem como operações entre dois vetores distintos. Para facilitar a interação, o programa disponibiliza um menu intuitivo que orienta o utilizador na execução das funcionalidades. Além das operações básicas, o projeto foi concebido com uma estrutura modular, garantindo organização e facilidade de manutenção do código. Esta abordagem contribui para a clareza e eficiência do programa, permitindo futuras expansões e melhorias. O trabalho visa consolidar os conhecimentos adquiridos na disciplina, aplicando conceitos fundamentais de programação estruturada e boas práticas de desenvolvimento.

Funcionalidades

  • Leitura e escrita de vetores
  • Ordenação e cálculo de mediana
  • Identificação de números primos
  • Construção e transposição de matrizes 18x18
  • Calculo da raíz quadrada
  • Devolução do vetor, com a primeira metade dos valores ordenada por ordem crescente
  • Cálculo da soma da primeira metade dos elementos no vetor com os da segunda metade (dá um vetor com metado do tamanho)
  • Devolução dos valores do vetor que não são divisíveis por três
  • Devolução de um vetor que mistura metade do primeiro vetor e metade do segundo
  • Cálculo e apresentação da matriz transposta
  • Apresentação de uma página de ajuda

Como usar

  1. Compilar com make
  2. Executar ./main.exe
